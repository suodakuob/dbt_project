{% docs order_status %}

Voici la définition des différents statuts de commande dans le Jaffle Shop :

| Status | Définition |
| :--- | :--- |
| **placed** | La commande a été créée mais pas encore expédiée. |
| **shipped** | Le colis est en route vers le client. |
| **completed** | Le client a reçu son colis avec succès. |
| **return_pending** | Le client a demandé un retour, en attente de réception. |
| **returned** | Le colis a été renvoyé et remboursé. |
